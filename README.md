# Weather App

A simple weather application that fetches real-time weather data using the Open-Meteo API. Users can search for any city and view current conditions plus a 5-day forecast.

Features
- Search weather by city name
- Displays current temperature, humidity, wind speed, and UV index
- 5-day forecast with weather icons
- Responsive design for desktop and mobile

Technologies Used
- HTML5
- CSS3
- JavaScript 
- Open-Meteo API (Geocoding + Forecast)
